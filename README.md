# onyx-spec

Clojure specs for Onyx core.

## Usage

In your project file:

```clojure
[org.onyxplatform/onyx-spec "0.11.0.3-SNAPSHOT"]
```

In your peer boot-up namespace:

```clojure
(:require [onyx.spec])
```
